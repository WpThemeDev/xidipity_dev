== Description ==
Xidipity is a meticulously crafted WordPress theme which brings new power to the classic editor and offers a beautiful sophisticated design supporting a wide range of solutions. Core to the theme is a robust editing solution containing twenty five drop down menu choices presenting over ninety editing options. Additional power is provided by the use of templates. Some of the more popular features include support of Font Awesome and Google material design icons - inclusion of Youtube, Vimeo, and HTML5 videos - and support for Google Adsense. Xidipity is designed to be responsive which ensures a positive user experience across many different devices. See this exciting theme in action at "https://xidipity.com/" or view the knowledge base at "https://doc.xidipity.com/".

== Installation ==

1. Download current release from GitHub at https://github.com/WpThemeDev/xidipity/releases
2. In the WordPress admin panel, go to Appearance -> Themes and click the 'Add New' button
3. Click upload theme button
4. Select the theme file
5. Click install now button
6. Click on the 'Activate' button to use your new theme right away
7. Configure as desired
8. Create content :)

== Disclosures ==

1. Xidipity uses advanced web technologies which may limit its ability to run on some browsers.
